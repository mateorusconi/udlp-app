# UDLP - Undemanding Device for Linguistic Processing

UDLP es un dispositivo digital diseñado para asistir a estudiantes con dificultades del lenguaje (dislexia, dispraxia, etc.) mediante una interfaz "no demandante" y herramientas de procesamiento lingüístico.

## Características (Versión 1.0)

- **Interfaz "Undemanding"**: Diseño limpio, colores suaves y tipografía clara.
- **Zona de Trabajo**: Editor de texto con controles visuales para ajustar tamaño de letra, espaciado e interlineado.
- **Barra Lateral de Consignas**: Muestra instrucciones claras y separadas del área de escritura.
- **Análisis Lingüístico (Prototipo)**:
  - Conteo de palabras en tiempo real.
  - Detección de oraciones muy largas.
  - Recordatorios de puntuación básica (mayúsculas).

## Tecnologías

- React
- Vite
- TailwindCSS

## Cómo ejecutar

1. Instalar dependencias:
   ```bash
   npm install
   ```
2. Iniciar servidor de desarrollo:
   ```bash
   npm run dev
   ```
3. Abrir en el navegador (usualmente `http://localhost:5173`).
